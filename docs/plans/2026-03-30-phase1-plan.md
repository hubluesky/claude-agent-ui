# Claude Agent UI — Phase 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the P0 core: one terminal inputs, all terminals see real-time sync. Read all projects and sessions with lazy loading.

**Architecture:** Monorepo (pnpm + turborepo) with 3 packages: shared (types), server (Fastify 5 + Agent SDK + WebSocket), web (React 19 + Vite + Tailwind v4 + Zustand). Server manages Agent SDK sessions, broadcasts via WS Hub, enforces single-input lock. Frontend lazy-loads projects/sessions via REST, real-time messages via WebSocket.

**Tech Stack:** Node.js 22+, @anthropic-ai/claude-agent-sdk, Fastify 5, @fastify/websocket, better-sqlite3, Drizzle ORM, React 19, Vite, TailwindCSS v4, Zustand

**Spec files:** `docs/specs/00-overview.md` through `docs/specs/08-phases.md`
**Design files:** `docs/design/01-desktop-chatting.png` through `docs/design/12-mobile-drawer.png`
