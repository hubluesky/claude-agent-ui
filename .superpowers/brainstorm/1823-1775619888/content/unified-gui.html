💡 核心思路

Tauri Rust 主进程统一管理所有子进程。不管是 tsx watch（开发）还是 sidecar exe（生产），对 GUI 来说都是"一个被管理的 Server 进程"。UI 层通过 Tauri Commands（Rust ↔ WebView 的 IPC）操作一切，用户永远不需要打开终端。

开发者工作流变成：双击打开 App → 选开发模式 → GUI 自动拉起 tsx watch + vite → 写代码 → 热更新即时生效
