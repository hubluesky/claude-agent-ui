对比总结

方案 A（推荐）：locked 用红色边框 + 锁图标在 placeholder 里，工具栏半透明。简洁明了
方案 B：locked 加红色半透明遮罩覆盖整个容器。视觉更强烈
方案 C：locked 在容器顶部加一条红色横幅。信息最清晰，但占空间
running 共同特征：橙色脉动灯 + "running" 文字 + 红色停止按钮 — 与 locked 完全不同的视觉语言
