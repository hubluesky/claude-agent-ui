核心布局范式：你心目中的"统一终端"是什么样的？

当前 UI 是经典的「侧边栏项目列表 + 单聊天窗口」模式。你希望往哪个方向进化？

A

IDE 风格：多标签页 + 侧边栏

类似 VS Code / Cursor。每个会话是一个标签页，可以同时打开多个会话快速切换。侧边栏保留项目/会话树。

B

Dashboard 风格：项目卡片 + 会话面板

首页是项目仪表盘，每个项目卡片显示最新状态/进度。点进去才是聊天。适合管理多个项目的全局视图。

C

混合风格：侧边栏仪表盘 + 标签页聊天

侧边栏变成带状态指示的项目/会话列表（显示运行状态、进度、待审批），主区域支持多标签页快速切换。结合 A 和 B 的优点。
