⑤ 技术架构 组件复用 + 独立 WS

组件复用策略

  • MessageComponent + compact prop
  • ChatComposer + minimal prop
  • ChatMessagesPane + limit prop
  • ApprovalPanel + compact prop
compact=true → 跳过语法高亮
minimal=true → 隐藏 toolbar
limit=20 → 只渲染最近 20 条

每面板独立 WebSocket

  • MiniChatPanel 创建自己的 WS
  • 独立 join session
  • 独立本地 state（不走全局 store）
  • 面板状态写入 multiPanelStore
服务端零改动
connectionStore 零改动
messageStore 零改动

模式切换 WS 交接

  • Single→Multi：主 WS leave
  • Multi→Single：面板 WS 断开
  • ↗ 展开：面板 WS → 主 WS
  • ← 返回：主 WS → 面板 WS
REST API 补回消息丢失窗口
面板 mount 时加载最近 20 条
