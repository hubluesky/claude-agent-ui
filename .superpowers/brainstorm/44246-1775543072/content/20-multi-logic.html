面板管理

  • 下拉菜单里 + 添加到面板
  • ✓ 标识已在面板中
  • 当前聊天会话自动在面板中
  • 面板列表持久化到 localStorage

Multi 显示逻辑

  • 显示所有已添加的面板
  • 最后一格 = 新建对话入口
  • ↗ 展开到 Single 深度聊天
  • × 关闭面板（从列表移除）

新建对话

  • Single：TopBar + 按钮（现有）
  • Multi：最后一个面板是新建入口
  • 选择项目 → 输入消息 → 新面板
  • 和 Single 新建完全一致的流程

模式切换

  • Single → Multi：显示所有面板
  • Multi ↗ → Single：全屏该会话
  • "← 返回 Multi" 或 Esc
  • 侧边栏始终不变
